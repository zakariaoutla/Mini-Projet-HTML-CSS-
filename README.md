## Project Description
This project is a faithful recreation of the Calm landing page based on the provided mockup (image), using **HTML and CSS only**.  
The page is fully **responsive** for both desktop and mobile devices, with accurate layout, colors, fonts, and spacing.

## Technologies Used
- **HTML5** — for structuring the page  
- **CSS3** — for styling, colors, fonts, spacing, and responsiveness

## Project Contents
- `index.html`  
- `style.css`  
- `images/` folder containing all images used

## Page Structure
1. **Header** — Calm logo, navigation links, and a Menu button  
2. **Hero Section** — Large image, main title, description paragraph, and call-to-action button  
3. **Services / Solutions Section** — Three columns, each with a title, description, and “Learn more” link  
4. **Catalog Section** — Rounded images aligned horizontally with consistent spacing  
5. **Testimonials Section** — Three user testimonials with gradient background and clear text  
6. **Offer Section** — Subscription plan, “14-day free trial” button, explanatory text, and main “Continue” button  
7. **Transition Image Section** — Large ocean image  
8. **Footer** — Four columns of links, legal info, and copyright
